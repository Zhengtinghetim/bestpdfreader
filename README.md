PDF Reader — Translate with Ease
A lightweight, browser-only tool that extracts text from PDF files and renders it as clean HTML — perfect for using Chrome's built-in translation to read foreign-language PDFs, including scanned books.
✨ Features

📄 PDF → HTML: Extracts and renders text with paragraph structure preserved
🖼 OCR support: Works on scanned books and image-only PDFs using Tesseract.js
🔍 Auto-detect mode: Tries text extraction first, falls back to OCR automatically
🌐 Chrome Translation ready: Once rendered, right-click → "Translate to [language]"
🔒 100% private: Everything runs in your browser — no server, no uploads
💾 Save as HTML: Download the rendered HTML file for later
📋 Copy text: One-click copy of all extracted text

Simply acccess: https://zhengtinghetim.github.io/pdf/

📖 How to Use

Open the website in Chrome
Select a mode (Auto-detect is fine if you're unsure):

🔍 Auto-detect — tries text extraction, switches to OCR if pages are blank
📝 Text PDF — fast, for regular digital PDFs
🖼 Scanned / OCR — for scanned books or image-only PDFs


Drop or select your PDF
Wait for extraction (OCR takes a few seconds per page)
Right-click → Translate to [your language] and read away!

🛠️ Tech

PDF.js by Mozilla — PDF parsing in the browser
Tesseract.js — in-browser OCR engine
Pure HTML/CSS/JS — zero build step, zero dependencies to install

⚠️ Limitations

OCR quality depends on scan quality — clean, high-res scans work best; blurry or skewed pages may have errors
Complex layouts (columns, tables) may have some text reordering
Password-protected PDFs are not supported
